# AGENTS.md

## Project

株式会社ルデラのコーポレートサイト。映像制作とWeb制作を扱う会社として、認知拡大と問い合わせ獲得を目的に制作する。

## Direction

- 先進的で遊び心のある表現を使う
- 法人担当者、経営者が安心して問い合わせできる情報設計にする
- 映像制作とWeb制作の両方を、成果につながるクリエイティブとして見せる
- ユーモアは強くしてよいが、問い合わせ導線と可読性は崩さない

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- Vercel

## Commands

- `npm run dev`: local development
- `npm run build`: production build
- `npm run lint`: lint check
